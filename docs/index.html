Markdown233

Native Markdown vault editor for Windows and macOS. Tauri 2, low bundle size, Git-first sync, command palette, settings center, graph view, plugin-ready core, and user-owned themes.

One-click install

powershell -ExecutionPolicy Bypass -File install.ps1
sh install.sh

Core surfaces

Use Ctrl/Cmd + K for the command palette. Right inspector panels switch between overview, outline, backlinks, graph, and theme.

Vault config

Each vault can keep its own .markdown233/config.json for theme, sync, graph layout, shortcuts, and plugin state.

Sync

Default sync strategy is Git. Local mirror sync supports cloud-drive folders such as iCloud Drive, OneDrive, Dropbox, or any user-selected directory.
