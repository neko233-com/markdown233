Theme Package

The theme studio imports and exports a stable JSON package. Users can keep, share, override, and restore themes without touching source code.

theme.json

{
  "name": "Calm Glass",
  "version": "1.0.0",
  "variables": {
    "accent": "#5b6cff",
    "surface": "#fbfcff",
    "text": "#1d1d26",
    "editorWidth": 860
  },
  "localeOverrides": {
    "newFile": "New note"
  }
}

Maintenance

Language overrides stay separate from built-in locale files. Users can reset overrides and return to the shipped translations at any time.
